# ADA final project

## Project Description
This repository includes the R markdown code for the ADA final project. 

## Files Included
- `ADA final project.Rmd`: R script used to summarize and visualize the data
- `README.md`: This file

## What the Code Does
- Download the BRFSS 2021 survey dataset from this link: https://www.cdc.gov/brfss/annual_data/annual_2021.html
- Reads in the survey dataset
- Performs basic data cleaning (e.g., renaming columns, filtering)
- Creates summary statistics (e.g., age, gender, responses)
- Generates a plot to visualize the data

## How to Run the Code
1. Download the BRFSS 2021 survey dataset from this link: https://www.cdc.gov/brfss/annual_data/annual_2021.html
2. Download or clone this repository to your computer
3. Open `ADA final project.Rmd` in RStudio
4. Make sure your working directory is set to the folder where the files are saved
5. Run the script
